# AI Mule Detection System

## Project Description
This project detects suspicious mule bank accounts using Machine Learning (Random Forest).

## Features
- Fraud detection prediction
- CSV upload support
- Cleaned data pipeline
- Download results

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
